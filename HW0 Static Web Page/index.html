สมุนไพรไทย 22 ชนิด เสี่ยงต่อการสูญพันธุ์


กระทรวงสาธารณสุขได้ประกาศรายชื่อ สมุนไพรไทย จำนวน 22 ชนิด ที่กำลังอยู่ใน “ภาวะเสี่ยงต่อการสูญพันธุ์”

เนื่องจากมีการใช้ทั้งต้น หรือใช้ทั้งราก ซึ่งทำให้พืชที่มีการนำมาใช้บ่อยๆนั้นเจริญเติบโตไม่ทัน กระทรวงสาธารณสุขจึงกล่าวว่า

ขณะนี้มีสมุนไพรไทยจำนวน 22 ชนิด ที่เสี่ยงอาจจะสูญพันธุ์ได้ ซึ่งประกอบด้วยประกอบไปด้วยสมุนไพร ดังต่อไปนี้

1. ถั่วดินโคก

ชื่อวิทยาศาสตร์ : Sophora exigua Craib

ลักษณะทางพฤกษศาสตร์ : ไม้พุ่มขนาดเล็ก ใบแบบขนนกเรียงสลับผิวใบมีขนละเอียดสีขาว ดอกแบบช่อกระจะที่ปลาย ก้านช่อดอกยาว ผลเป็นฝักรูปขอบขนาน

สรรพคุณ : รากฝนกับน้ำดื่มช่วยลดไข้ ฝนทาแก้พิษงู บำรุงน้ำนม ขับพิษภายใน แก้ฟกบวมตามข้อและกล้ามเนื้อ แก้คางทูม

2. เทพทาโร

ชื่อวิทยาศาสตร์ : Cinnamomum porrectum (Roxb.) Kostern

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นขนาดกลางถึงใหญ่ เรือนยอดโปร่งถึงเป็นพุ่มกลมทึบ เปลือกต้นสีเทาอมเขียวหรือเทาอมน้ำตาลแตกเป็นร่องตามยาวลำต้น

สรรพคุณ : แก้จุกเสียด ขับลม เปลือกเป็นยาบำรุงร่างกาย แก้ปวดท้อง ขับลม ท้าให้เรอ

3. มะตูมนิ่ม

ชื่อวิทยาศาสตร์ : Aegle marmelos (L.) Correa

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นขนาดเล็ก เปลือกสีเทาเรียบ และมีกลิ่นหอม ดอกออกเป็นช่อที่ปลายกิ่ง ผลรูปรีกลมผิวเรียบเกลี้ยงเปลือกหนา

สรรพคุณ : ใช้เป็นยาอายุวัฒนะ ขับผายลม บำรุงกำลัง กินผลสุกเป็นยาแก้พิษฝี พิษไข้ รักษาน้ำดี

4. มะหาด

ชื่อวิทยาศาสตร์ : Artocarpus lakoocha Roxb

ลักษณะทางพฤกษศาสตร์ : ไม้ต้นขนาดกลางถึงใหญ่ เรือนยอดเป็นพุ่มกลมทึบ เปลือกมีสีเทาแกมน้ำตาล ตามยอดมีขนนุ่มสีเทาถึงน้ำตาลปกคลุม

สรรพคุณ : แก้ไข้ แก้จุกเสียดแน่น ท้องอืดท้องเฟ้อ ขับลม ขับโลหิต เป็นยาระบาย ถ่ายพยาธิ

5. เร่ว

ชื่อวิทยาศาสตร์ : Amomum villosum Lour

ลักษณะทางพฤกษศาสตร์ : เป็นพืชล้มลุก มีเหง้าหรือลำต้นอยู่ในดิน ใบยาวเรียวปลายใบแหลม ออกดอกเป็นช่อจากยอดที่แทงขึ้นมา ผลมีขนสีแดงปกคลุม

สรรพคุณ : ใช้ขับลม แก้ท้องอืดเฟ้อ จุกเสียด ลำต้นใต้ดินมีกลิ่นหอมนำมาใช้เป็นเครื่องเทศ

6. หัวร้อยรู

ชื่อวิทยาศาสตร์ : Hydnophytum formicarum Jack

ลักษณะทางพฤกษศาสตร์ : ไม้พุ่ม อิงอาศัยบนต้นไม้อื่น ลำต้นสูง อวบน้ำ ภายในเป็นโพรง ผิวใบเกลี้ยงปลายใบมน ดอกสีขาวออกเป็นกระจุก

สรรพคุณ : หัวบำรุงหัวใจ ขับชีพจร ขับพยาธิ แก้พิษในข้อในกระดูก แก้พิษประดง แก้ข้อเข่าข้อเท้าบวม

7. กระทุ่มนา

ชื่อวิทยาศาสตร์ : Mitragyna diversifolia (Wall. ex G.Don) Havil

ลักษณะทางพฤกษศาสตร์ : ไม้ต้นขนาดกลาง เรือนยอดเป็นพุ่มกลม ใบรูปไข่หรือรูปรีที่ค่อนข้างกว้างเกือบกลม ปลายมน ขอบเรียบ

สรรพคุณ : ลดความดันโลหิตและออกฤทธิ์กดประสาทและกล้ามเนื้อในสัตว์ทดลอง เปลือกต้นรสฝานร้อน รักษาโรคผิวหนัง

8. ขันทองพยาบาท

ชื่อวิทยาศาสตร์ : Suregada multiflora (A.Juss) Baill

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นขนาดกลาง มีทรงพุ่มแน่นทึบ ลำต้นตั้งตรง เปลือกต้นมีสีน้ำตาลแก่ ผิวบางเรียบ ใบเป็นรูปขอบขนานแกมรูปหอก

สรรพคุณ : แก้ลม แก้ประดง แก้พิษในกระดูก แก้โรคผิวหนัง รักษาน้ำเหลืองเสีย

9. จุกโรหินี

ชื่อวิทยาศาสตร์ : Dischidia major (Vahl) Merr

ลักษณะทางพฤกษศาสตร์ : ไม้ล้มลุกที่มีลำต้นเลื้อยทอดไปตามต้นไม้ เถากลมสีเขียว ตามข้อเถามีรากมีไว้สำหรับใช้ยึดเกาะ ออกผลเป็นฝัก

สรรพคุณ : ต้มน้ำดื่ม แก้ปวดท้อง แก้หอบหืด ใช้เป็นยาฝาดสมาน แก้ท้องร่วง

10. ชะเอมไทย

ชื่อวิทยาศาสตร์ : Albizia myriophylla Benth

ลักษณะทางพฤกษศาสตร์ : ไม้เถาขนาดใหญ่ มีหนาม ใบประกอบแบบขนนกสองชั้น ใบเล็กละเอียดเป็นฝอย ดอกช่อออกตามปลายกิ่งลักษณะเป็นพู่

สรรพคุณ : แก้โรคในลำคอ บำรุงธาตุและบำรุงกำลัง รากมีรสหวานทำให้ชุ่มคอแก้กระหายน้ำ มีสรรพคุณแก้ไอ

11. ชิงชี่

ชื่อวิทยาศาสตร์ : Capparis micracantha

ลักษณะทางพฤกษศาสตร์ : เป็นไม้พุ่มหรือรอเลื้อยมีหนาม ใบรูปขอบขนานและรูปใบหอก ดอกออกเป็นช่อสั้นๆมีขนสั้นนุ่ม

สรรพคุณ : ช่วยให้มดลูกเข้าอู่ เป็นยาบำรุงหลังคลอดบุตร แก้ไอเนื่องจากหลอดลมอักเสบ ขับลม

12. ตับเต่า

ชื่อวิทยาศาสตร์ : Diospyros ehretioides Wall. ex G.Don

ลักษณะทางพฤกษศาสตร์ : ต้นเป็นทรงพุ่มโปร่งเป็นรูปกรวย เปลือกต้นเป็นสีน้ำตาลปนเทา ใบเป็นรูปไข่กว้าง เนื้อใบเกลี้ยงและหนา

สรรพคุณ : ช่วยบำรุงสุขภาพ บำบัดอาการปวดหลังปวดข้อ ปวดเส้นเอ็นและกระดูก

13. นางแย้มป่า

ชื่อวิทยาศาสตร์ : Clerodendrum viscosum Vent

ลักษณะทางพฤกษศาสตร์ : เป็นไม้พุ่มขนาดย่อม ลำต้นตั้งเป็นสันสี่เหลี่ยมตรงไม่มีขนปกคลุม ดอกออกเป็นช่อบริเวณปลายกิ่งเป็นกลุ่ม

สรรพคุณ : รักษาลำไส้อักเสบ ช่วยขับปัสสาวะ แก้ไตพิการ เป็นยาแก้ไข้ บำรุงน้ำนมของสตรี

14. ปลาไหลเผือก

ชื่อวิทยาศาสตร์ : Eurycoma longifolia Jack

ลักษณะทางพฤกษศาสตร์ : เป็นไม้ลงราก รากกลมโตสีขาวยาว กิ่งก้านสั้นเป็นกระจุกที่ปลายยอดของลำต้น รากจะมีรสขม

สรรพคุณ : ใช้รากต้มน้ำดื่ม แก้ไข้ แก้ปวดท้อง ทำเป็นยาบำรุงกำลัง ใช้เลิกเหล้า

15. พังคี

ชื่อวิทยาศาสตร์ : Croton crassifolius Giesel

ลักษณะทางพฤกษศาสตร์ : ไม้พุ่มขนาดเล็ก มีรากขนาดใหญ่ใต้ดิน มีกลิ่นหอม โคนใบมนปลายแหลม กลีบดอกสีขาวหรือสีเหลือง

สรรพคุณ : รากใช้ต้มน้ำดื่ม แก้ท้องอืด ท้องเฟ้อ หรือตำประคบเพื่อแก้ปวด แก้ฟกช้ำ

16. มะคังแดง

ชื่อวิทยาศาสตร์ : Gardenia erythroclada Kurz

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นกึ่งไม้พุ่ม ต้นและกิ่งก้านสีน้ำตาล มีขนนุ่มๆสีน้ำตาลแดงปกคลุม โคนต้นและกิ่งมีหนามโดยรอบ เนื้อไม้มีสีขาวนวล

สรรพคุณ : ต้มน้ำดื่ม แก้เลือดลมเดินไม่สะดวก เปลือกต้นตำพอกแผลสด ห้ามเลือด รากใช้เป็นยาถ่าย

17. สะค้าน

ชื่อวิทยาศาสตร์ : Piper ribesoides Wall

ลักษณะทางพฤกษศาสตร์ : ไม้เถาเลื้อย ลำต้นอวบอ้วนใหญ่ เนื้อใบเหนียวและหนามาก เปลือกค่อนข้างอ่อน เนื้อสีเหลืองน้ำตาล

สรรพคุณ : รสเผ็ดร้อน ขับลมในลำไส้ แก้จุกเสียด ใช้เป็นยาบำรุงกำลัง

18. สารภีป่า

ชื่อวิทยาศาสตร์ : Anneslea fragrans Wall

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นขนาดเล็กถึงขนาดกลาง ใบหนาเรียบและเป็นมัน ดอกเดี่ยวมีสีเหลืองแกมชมพู

สรรพคุณ : ใช้ถ่ายพยาธิ แก้บิด แก้ไข้ ต้มน้ำดื่มรักษาโรคประดง

19. อบเชยไทย

ชื่อวิทยาศาสตร์ : Cinnamomum bejolghota (Buch.-Ham.) Sweet

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นเรือนยอดเป็นพุ่มกลม ดอกเป็นช่อแบบช่อกระจุกสีเหลืองอ่อน

สรรพคุณ : เปลือกแก้อ่อนเพลีย ชูกำลัง บำรุงธาตุ รากและใบต้มดื่มแก้ไข้จากการอักเสบหลังคลอด ขับลม

20. เฉียงพร้านางแอ

ชื่อวิทยาศาสตร์ : Carallia brachiata (Lour.) Merr

ลักษณะทางพฤกษศาสตร์ : ไม้ยืนต้นเรือนยอดทรงพุ่ม ดอกช่อแยกแขนง ผลแก่มีสีส้มปนแดง

สรรพคุณ : แก้ร้อนในกระหายน้ำ แก้ไข้ ขับเสมหะและโลหิตสมานแผล แก้พิษผิดสำแดง บำรุงร่างกาย

21. เถาเอ็นอ่อน

ชื่อวิทยาศาสตร์ : Cryptolepis buchanani Roem.& Schult

ลักษณะทางพฤกษศาสตร์ : ไม้เลื้อยพาดต้นไม้อื่น เปลือกเถาเรียบสีน้ำตาล ต้นมีน้ำยางสีขาว ใบเรียบเป็นมันและลื่น

สรรพคุณ : เถานำมาต้มดื่มบำรุงเส้นเอ็นให้แข็งแรง แก้ปวดเมื่อยเส้นเอ็น แก้ขัดยอก ทำให้คลายการตึงตัว

22. เปราะหอม

ชื่อวิทยาศาสตร์ : Kaempferia galanga linn

ลักษณะทางพฤกษศาสตร์ : เป็นพืชล้มลุก มีลำต้นใต้ดิน ใบกลมเป็นรูปไข่ป้อม ดอกมีสีขาวแต้มด้วยสีม่วงเหลือง

สรรพคุณ : รสเผ็ดขม แก้หวัดคัดจมูก ขับลม แก้เสมหะ แก้โลหิตซึ่งเจือด้วยลมพิษ


นอกจากนี้ ยังมีการเลือกสมุนไพร 6 ชนิดจาก 22 ชนิดขึ้นมา และมีการเสนอสมุนไพร กระวาน ขึ้นมาเป็นพิเศษด้วยเพิ่มอีกชนิดหนึ่ง

เพื่อทำการประกาศคุ้มครอง และให้เป็นสมุนไพรที่ต้องอยู่ในการควบคุม ทั้งหมด 7 ชนิด ได้แก่ ถั่วดินโคก เทพทาโร มะตูมนิ่ม มะหาด เร่ว หัวร้อยรู และกระวาน
